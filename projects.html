Image Gallery


Snap Shot is a modern, visually striking web-based image gallery designed to showcase multimedia content through a clean and structured card-based user interface. Built entirely using HTML5 and CSS3, the project features an immersive, dark-themed aesthetic accented by vibrant neon gradients that create a sleek, futuristic backdrop. The layout elegantly organizes content into distinct vertical cards, each featuring a framed high-resolution image, a bold identifier, an informative description, and a "Learn more" call to action. Leveraging advanced CSS layout techniques like Flexbox or Grid, the design ensures uniform alignment, polished spacing, and rounded borders that elevate the overall UI/UX. This project highlights a strong command of semantic web structure, clean typography, and visual hierarchy, making it an excellent foundation for a responsive portfolio or dynamic media directory.

Dynamic Image Slider


Developed as a modular front-end web application, the Dynamic Image Slider is a responsive UI component built with ReactJS, HTML, CSS, and JavaScript that seamlessly balances automation with precise user control. The project features an intelligent dual-navigation carousel engine that automatically cycles through image assets every 3 seconds while smoothly integrating manual next and previous navigation overrides through reactive state management. Designed with a mobile-first approach, the architecture utilizes custom CSS media queries and modern layout principles to ensure fluid cross-device adaptability across smartphones, tablets, and desktops without relying on heavy external UI frameworks. By focusing on component reusability and clean execution, this project effectively demonstrates core competencies in semantic frontend structuring, lifecycle management, and lightweight, high-performance web development, making it an excellent plug-and-play solution for modern web ecosystems.
